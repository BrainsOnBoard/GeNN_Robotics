Get an estimate for heading based on comparing image with stored snapshots.

The parameters are perfect-forwarded to the Rotater class, so e.g. for InSilicoRotater one passes in a cv::Mat and (optionally) an unsigned int for the scan step and for the AntWorldRotater, one passes in one or more angles.
